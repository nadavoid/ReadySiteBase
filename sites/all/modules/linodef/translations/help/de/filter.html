Nach der Installation von Linodef zur Seite Verwaltung → Einstellungen → Eingabeformate wechseln und Linodef im bevorzugten Eingabeformat aktivieren. Nähere Hinweise zur Benutzung der Linodef Tags inklusive Beispiele finden sich auf der Hilfeseite zu den Eingabeformaten.
Layout von eingebundenen Feldern
Eingebundene Felder besitzen das gleiche Layout wie im originalen Node, d.h. alle eingestellten Filter des standardmäßigen Eingabeformats werden berücksichtigt. Mit einer Ausnahme: Um eine unendliche Rekursion zu vermeiden, werden Linodef tags entfernt. Dieser Fall kann auftreten, wenn sich Felder gegenseitig oder sich selbst mittels Linodef Tags referenzieren.

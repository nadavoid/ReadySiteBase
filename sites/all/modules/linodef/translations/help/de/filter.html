Nach der Installation von Linodef zur Seite Verwaltung → Einstellungen → Eingabeformate wechseln und Linodef im bevorzugten Eingabeformat aktivieren. Weiterhin muss Linodef als erstes vom Eingabeformat ausgeführt werden. Dazu die Sortierungsseite aufrufen (admin/settings/filters/*/order) und Linodef nach oben schieben (so dass es die kleinste Zahl in der Stellenwertanzeige besitzt). Nähere Hinweise zur Benutzung der Linodef Tags inklusive Beispiele finden sich auf der Hilfeseite zu den Eingabeformaten.
Layout von eingebundenen Feldern
Eingebundene Felder besitzen das gleiche Layout wie im originalen Node, so lange das dortige Eingabeformat mit dem des einbindenden Feldes übereinstimmt oder die Unterschiede der Eingabeformate nicht das Layout beeinflussen. Allerdings gibt es eine Ausnahme: Um eine unendliche Rekursion zu vermeiden, werden Linodef tags entfernt. Dieser Fall kann auftreten, wenn sich Felder gegenseitig oder sich selbst mittels Linodef Tags referenzieren.

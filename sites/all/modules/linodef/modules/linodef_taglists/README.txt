Linodef - Taglists
------------------
This module provides an user interface to create lists of drupal objects and
attach Linodef tags to them.

Those lists can be used by modules which provide editor buttons, blocks and
other content.

Go to Administer -> Site building -> Modules and enable "Linodef taglists".

Usage
-----
To access the taglist setup go to admin/build/taglists.

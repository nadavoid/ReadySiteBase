This function retrieves the data for linodef_buttonsapi_itemlist. Its return array does not have the content key so if your button does not need a formatted list use this function.

Name

linodef_buttonsapi_rawdata

Parameter

string $drupal_element
See get formatted items.
string $option_string_path
See get formatted items.

Return

  • If the data retrieving is successful, than the [content] key is not part of the this array. Besides that it is similiar to the array get formatted items returns.
  • If the data could not be obtained, a string is returned with an info message containing the reasons.

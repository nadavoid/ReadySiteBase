Linodef bietet die Schaltflächen API für diejenigen an, die die Linodef Schaltflächen in ihrem bevorzugten Editor nutzen wollen, der aber bisher noch nicht durch ein Untermodul unterstützt wird. Vorzugsweise sollte dabei das Linodef - BUEditor Modul als Vorlage genutzt werden (den Untermodulordner kopieren & umbennenen), um sich vollkommen auf die benötigten Anpassungen der .module Datei sowie der Hilfedatei im Ordner "help" konzentrieren zu können. Bei generellen Fragen zu Modulen in Drupal, bitte die Seiten Modulerstellung in Drupal 6.x im Handbuch zu Rate ziehen. Die folgenden Dinge müssen beim Erstellen eines Editorunterstützungsmoduls beachtet werden:
  1. Die JavaScript Datei, die das onClick Ereignis enthält muss den Namen modulname_functions.js tragen und im Wurzelverzeichnis des Moduls liegen.
  2. Die JavaScript Funktion für das onClick Ereignis muss den Namen modulname_onclick tragen und einen String Parameter für den Linodef Tag bereitstellen.
Falls die Dokumentation nicht ausreichen sollte, so sind detaillierte Informationen über die einzelnen Funktionen im API Modul selbst zu finden.

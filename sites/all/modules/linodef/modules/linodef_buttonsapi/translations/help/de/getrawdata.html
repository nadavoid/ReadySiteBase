Diese Funktion ermittelt die Daten für linodef_buttonsapi_itemlist. Der Rückgabearray hat keinen content Schlüssel. Falls eine Schaltfläche keine formatierte Liste benötigt, so ist diese Funktion vorzuziehen.

Name

linodef_buttonsapi_rawdata

Parameter

string $drupal_element
Siehe Formatierte Listen.
string $option_string_path
Siehe Formatierte Listen.

Return

  • Falls die Datenermittlung erfolgreich war, wird der Schlüssel [content] nicht im Array mit zurückgegeben. Davon abgesehen ist der Array identisch zum Rückgabearray der formatierten Listen.
  • Falls keine Daten erstellt werden konnten, wird ein String zurückgegeben, der Informationen über die Ursachen enthält.

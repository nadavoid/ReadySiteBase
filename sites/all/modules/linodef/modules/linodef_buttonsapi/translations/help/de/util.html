Im Folgenden ist eine Liste nützlicher Hilfsfunktionen zu finden:
linodef_buttonsapi_validate_tagoptions($string)
Erstellt ein Array von Linodef Tag Optionen aus einem String, in dem die Optionen per "_" oder "," getrennt sind (linodef_buttonsapi.module).
linodef_buttonsapi_tagoptions_to_path($array)
Konvertiert einen Linodef Tag Optionen Array in einen Pfad-kompatiblen String (linodef_buttonsapi.module).

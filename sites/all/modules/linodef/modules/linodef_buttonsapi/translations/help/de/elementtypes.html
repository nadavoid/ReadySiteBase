Die Datenermittlungsfunktion der Linodef Schaltflächen API filtert den Parameter nach dem Elementtypen, der beim Erstellen der Editorschaltfläche eingegeben wurde. Die folgende Liste zeigt unterstützte Elementtypen sowie die Elemente, die die Schalflächen API daraufhin ausgibt:
Feldname
Die Werte der Felder mit diesem Namen sowie die eingeklammerten Titel der zugehörigen Nodes, d.h. Feldname (Nodetitel).
Inhaltstyp
Die Titel der Nodes dieses Inhaltstypes.
Vokabular ID
Alle Taxonomiebegriffe, die Teil des Vokabulars sind.
Dabei ist zu beachten, dass auch die Vokabular ID als String eingegeben werden muss.
Sicherheitshinweise
Falls der aktuelle Benutzer keine Zugriffsrechte zu einzelnen Nodes besitzt, so tauchen diese bei Anzeige der Schaltfläche für den Benutzer auch nicht auf. Bei Verwendung von Modulen Dritter, die in das Zugriffsmanagement von Drupal eingreifen und dabei ausversehen die Funktion node_access() umgehen, kann es unter Umständen zu unbeabsichtigten Ergebnissen kommen (Nodes, auf die der Nutzer keinen Zugriff hat, tauchen trotzdem in der Liste auf oder andersherum). In diesem Fall bitte einen Eintrag in den Linodef "issue queue" vornehmen und dabei das benutzte Modul benennen sowie ein kurze Beschreibung der Symptome verfassen.
